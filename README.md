## About

Music player with library management and streaming support.

## Features

- Library management
- Playlists
- Queue
- Equalizer
- Lyrics

## Installation

### From App Store
Open the **App Store** in zOS and search for "Music".

### Via npm
```bash
npm install @zos-apps/music
```

## Usage

- Add music to library
- Create playlists
- Control playback at bottom
